Extension Methods Sample

Purpose

The sample shows how to implement and call custom Extension Methods.

Examine the code

Extenson Methods enable you to "add" methods to existing types without creating a new derived type, recompiling, or otherwise modifying the original type.
Defining an extension method is done by applying an [Extension] attribute to a static class and its class method.
This attribute must be defined in the System.Runtime.CompilerServices namespace.
You have several ways to get the Extension Attribute class:
  • If you're sure that .NET 3.5 is installed on the target machine, you can reference the .NET frameworks's System.Core.dll. But remember: You don't need this version of the framework for extension methods.
  • You can take the System.Core.dll of the Mono framework, which runs on .NET 2.0, too.
  • You can define it on your own in a seperate assembly.
From Program.pas you can see extended with new methods Integer type and IEnumerable interface.
