Sequences and Queries Sample

Purpose

The sample shows features of Oxygene related to sequences and Language Integrated Query (LINQ) statements.

Examine the code

Sequences are a special type introduced by the Oxygene language, and can be thought of as a collection of elements, similar to an array.
Examine code (Main.pas) to see how to use different query expressions for retrieving data from sequences.
  • In order to use the Standard Query Operators, is needed reference to the System.Core.dll and the System.Linq namespace in your uses clause.
  • All query expressions are started with a from clause, and can be followed by zero or more additional clauses that act on the sequence.

Getting started

  • Compile the entire solution.
  • Run the application.
  • Try to execute different query statements to see the result in the grid below.
